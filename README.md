# VirtAPU
A virtual Audio Processing Unit to simulate 8-bit era game sounds 

 - 4 voices:
   - Pulse
   - Triangle
   - Noise1 (LFSR 1)
   - Noise2 (LFSR 6)
 - Pulse width modulation
 - Decay
 - Frequency sweep
 - Sequencer
